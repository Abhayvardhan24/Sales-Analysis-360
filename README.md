Project Objectives - Understand sales across products, categories, stores, regions. - Identify growth opportunities and inefficiencies. - Highlight customer purchase patterns. 
Tools & Technologies Used - SQL, Python, Pandas, NumPy, Matplotlib, Seaborn. 
Data Sources - Orders, Product Master, Customer Master, Store Master, Returns.
Key KPIs - Sales, Margin%, GMV, AOV, Units Sold, Repeat Rate, Return Rate, Growth.
Business Insights - Seasonal spikes, top products drive majority revenue, regional decline, promo customers 
churn faster.
